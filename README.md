# おすすめdiscord BOT ４選！

Discordサーバーを運営するときに便利なおすすめBOTを5つ紹介します。サーバーの目的（管理、音楽、盛り上げなど）によって組み合わせるといい感じです。

1. **MEE6**
   　多機能な管理（モデレーション）BOTとして定番。レベル（XP）システム、ウェルカムメッセージ、自動モデレーション、カスタムコマンドなどをサポート。 ([discord-media.com][1])

2. **Carl-Bot**
   　役職（ロール）管理が得意。リアクションでロールを付与したり、ログ記録、カスタムコマンドなど多用途に使える管理BOT。 ([discord-media.com][1])

3. **Hydra**
   　音楽BOT。YouTubeやSpotifyなど複数の音源に対応し、ウェブコントロールが可能。 ([nanchara.jp][3])

4. **Dank Memer**
   　ミーム系や遊び系が得意なBot。通貨システム、ミーム生成、ゲームっぽい要素があって、サーバーを盛り上げるのに向いている。 ([appeio.com][2])
